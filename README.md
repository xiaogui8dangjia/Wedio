## 技术架构

### 核心类
1. **MainWindow** - 主窗口类，管理应用程序界面
2. **ThreeDCanvas** - 3D绘图区类，负责OpenGL渲染
3. **Camera** - 相机类，管理3D视图变换

### 关键技术
- **GUI框架**: Qt 6 (C++)
- **3D渲染**: OpenGL 3.3 + Qt3D
- **数学库**: Qt自带数学库（QVector3D, QMatrix4x4, QQuaternion）
- **构建系统**: CMake + qmake
- **代码结构**: 模块化设计，便于扩展
